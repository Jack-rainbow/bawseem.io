# 字符串

# charAt

  • 1.小记

charAt返回字符串自身,如果未提供索引则返回0.类似 'a'.charAt()//此时返回0, 'b'.charAt(0)//返回b自身。 说白了就是 大写返回大写，小写返回小写

  • 2.实现方法
String.prototype.charAt = function (index){
    let args = this;
    if(index){
        for(let i =0;i<args.length;i++){
            return args[index];
        }
    }else{
        return this.toString();
    }
}

# split

  • 1.小记

将字符串分割成数组

  • 2.实现方法
String.prototype.split = function (val) {
    let arr = [];
    let _this = this;
    if (val) {
        arr.push(_this.toString())
    } else {

        for (let i = 0; i < _this.length; i++) {
            arr.push(_this[i])
        };
    }
    return arr;

}

# toString

  • 1.小记

返回字符串

  • 2.实现方法
String.prototype.toString = function(){
    let _this = this;
    let str = '';
    for(let i =0;i<_this.length;i++){
        str += _this[i];
    }
    return str;
}
