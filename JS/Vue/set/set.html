# 侦听数组和对象改变

🐱

侦听对象的添加与删除

平常写法
    eg:
        Object.assign({},this.xxx,{
            age:11,
            name:yuncl
        })
vue写法
    eg：
        //第一个参数
        this.$set(this.number,index,++this.number[index]);

已经创建的实例,vue不能动态添加根的响应式属性

data:{
    info:{
        age:222
    }
}
//第一个参数--data中的info变量
//第二个参数--自定义变量名
//第三个参数--自定义变量名的参数
vm.set(vm.info,'sex',女)

侦听数组的改变

vue不能检测一下变动的数组
    1.利用索引值设置项时
        vm.items[index] = newValue;
      解决方法：
        1. vue.set
            //有迷惑
            //第一个参数 是组件内的items
            //第二个参数为索引值
            //第三个参数为新的属性
            vm.set(example1.items,index,newvalue);
        2.Array.prototype.slice
            example1.items.slice(index,1,newvalue);
    2.修改数组的长度时
        vm.items.length = newLength;
       解决方法：
        1.splice--生成一个新的数组
            //example1.items 组件内的items
            //有迷惑
            example1.items,splice(newLength);
