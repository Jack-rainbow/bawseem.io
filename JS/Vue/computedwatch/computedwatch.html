# computed与watch区别

1.computed会在DOM加载后马上加载
2.watch 观察和响应 Vue 实例上的数据变动，对应一个键和对象
值是对应的回调（方法、对象、选项),
数组的长度(vm.set(this.items.splice(newlength)这样可以侦听到)
不可以通过watch侦听到,不使用箭头函数以防改变this的指向	

watch使用方式	//反应有时间间隔,大概1-2s

1.v-model来侦听数据变化
v-model="MobilePhone"
data中MobilePhone=""
watch: {
    	MobilePhone:function(val) {
       	    thisVal = $.trim(val),
      	    thisValLen = thisVal.length;
      	    if (isNaN(Number(val)) || thisVal == '' || thisValLen < 11) {
            	this.ntifyingCode = "";
       	 } else if (thisValLen == 11){
        	    this.ntifyingCode = "ntifyingCode";
       	 }
    }

2.vue.$watch('abc', function (val)=>{
	console.log(val)
})

<input  v-model="abc" />
data:{
  return {
    tdArray:"张三"
  }
},
watch:{
	abc(val){
	},
	b: 'someMethod',//值是方法
	c: {
	    //值是对象
  		    handler: function (val, oldVal) { /* ... */ },
 		    //深层拷贝watch
            deep: true
        }
}

3.直接侦听某个值
html中写Alipay，在data中写一个Alipay，然后在watch中写判断
watch: {
    Alipay，(val) {
        if (isNaN(Number(val)) || $.trim(val) == '') {
            this.Alipay  = " "
        } else {
            this.Alipay  = "ntifyingCode"
        }
    }
}	

执行顺序一般为computed--》watch
如果是触发事件后先methos--》watch
