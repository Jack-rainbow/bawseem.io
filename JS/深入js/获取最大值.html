# 获取数组中最大的值

Math.max 因为math.max参数只接受字符串，所以此处应使用apply

Math.max.apply(null,[]) //只适用于一位数组

//因为在js中每一个Functin()函数都有call,apply方法
//故Math.max()是函数，所以拥有此方法，所以利用此特性进行求解最大值


//多维数组求最大值

Math.max.apply(null,[[[]],[]].join(',').split(','))

let arr = [1,2,[3,[123],1],1,[2,[3,4,[5],6],4,3],512];

# toString和toLocalString的区别

|传入的值|	toString|	toLocaleString|
| ------ | ------ | ------ |
|数字|	字符串|	字符串|
|字符串|	字符串|	字符串|
|数组|	字符串|	字符串|
|普通对象||	“[object Object]”|	“[object Object]”|

正常用法上边都一样，但是当转换时间对象时，会出现问题
eg：
    (new Date).toString();
    <!--"Mon Nov 06 2017 13:02:46 GMT+0800 (China Standard Time)"-->

    (new Date).toLocaleString();
    <!--"2017/11/6 下午1:03:12"-->
同时千分位分割可以利用toLocalString()特性进行分割
    (12312313123353).toLocalString('en-US')

# != 与 !==区别

1.比较结果时
    != 返回同类值比较结果
    !== 不同类型不比较，并且无结果，只有当类型相同时才比较
2.比较过程上的区别
    != 比较时,若类型不同时，先转换类型
    !== 只有类型相同时才比较

# Object.setPrototypeOF 实现

Object.setPrototypeOf() 将第一个原型里边的_proto_指向另一个对象
第一个参数是需要修改原型的对象
第二个参数是修改到原型的原型对象(可能是一个空对象)

实现方法:
Object.setPrototypeOf = function(obj,proto){
    obj._proto_ = proto;
    return obj
}

子类的原型的原型，是父类的原型  子类.__proto__.__proto__ == 父类.__proto__
