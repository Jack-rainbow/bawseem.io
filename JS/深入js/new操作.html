# new的过程中操作了什么？

    var a = new Person('x','1');
    new Person  {
        var obj = {};
        //创建一个对象
        obj._proto_ = Person.prototype; 
        // 将新创建的空对象的隐式原型指向其构造函数的显示原型。
        var result  = Person.call(obj,arguments);
        //Person继承obj上边的参数也就是实参，并且改变this指向(因为call，apply，bind具有改变this的功能)
        return typeof result === 'obj' ? result  :  obj;
        //如果返回值是一个新对象的话那么直接直接返回该对象否则就返回形参
    }

# 普通函数与new 函数的区别？

    function abc(){
        let a = '1';
        console.log(this.a);
        console.log(a);
    }

普通函数的调用abc(),返回结果是undefined(this指向了最后调用他的那个对象),1因为他里边的this是指向了最后调用他的那个对象
当函数 new abc()之后,返回结果是undefined,1，因为此时new 之后函数返回的是实例对象，此时的实例对象里边没用this.a 所以返回undefined，里边的this指向了实例对象
