    # 内存管理

    ​

    1.简介

    JS 在创建变量时, 自动分配内存, 并在不使用的时候, 自动释放 。然后在用户不再使用时就会“自动”释放。所以此操作称为`垃圾回收。

    2.存储

    1.基础数据类型

    基本类型是一种既非对象也无方法的数据。共有6种类型：string,number,boolean,null,undefined,symbol。基础数据类型都存储在栈中，同时遵循先进后出的规则。

    ​

    1.1 判断基础数据类型

    typeof

    2.引用类型

    引用类型是对象且有方法的数据。共有3中类型: Array,Function,Object.引用类型都存储在堆中，存储在栈中的只是一个指针,同时遵循平行规则（即取即拿）。

    ​

    2.1  判断引用类型

    instanceof || Object.prototype.toString.call(type)

    The Sample 1:

    
var a1 = 0;   // 变量对象

var a2 = 'this is string'; // 变量对象

var a3 = null; // 变量对象
 
var b = { m: 20 }; // 变量b存在于变量对象中，{m: 20} 作为对象存在于堆内存中

var c = [1, 2, 3];// 变量c存在于变量对象中，[1, 2, 3] 作为对象存在于堆内存中
