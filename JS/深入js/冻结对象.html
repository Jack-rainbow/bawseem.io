# Object.seal与freeze区别

1.Oject.seal是封印对象，但是封印完对象还可以对对象进行读写
  Object.freeze是冻结对象，冻结后的对象不可以进行读写，只可以进行查看
2.Object.isExtensible()查看对象是否可以扩展
3.Object.isSealed()查看对象是否封印
  Object.isFrozen()查看对象是否冻结
