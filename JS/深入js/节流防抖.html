# 节流防抖

1.说明

节流（throttle）在一段时间内只执行一次，应用场景：按钮单击、input输入等
防抖(debocunce)在间隔多少秒后才执行，如果在这期间有继续单击的话会重新计时，应用场景：搜索输入框,window改变resize的时候

最终目的：为了解决频繁触发某个事件而造成不必要的性能浪费

2.节流

const throttle = (fn,ms = 1000) => {
    let isClick = true;
    return function(){
        if(!isClick)return;
        isClick = false;
        setTimeout(()=>{
            fn.call(this,arguments);
            isClick = true;
        },ms)
    }

}

3.防抖

const debounce = (fn,ms = 1000) => {
    let isClick = null;
    return function(){
        clearTimeout(isClick);
        isClick  = setTimeout(()=>{
            fn.call(this,arguments);
        },ms)
    }
}
