# 求平均数

1.利用map和reduce机制

const arr = (acc,val)=>{
    arr.map(typeof val === 'function '? val:val=>{
        return acc[val];// 如果acc是{},返回他的键值并且利用map可以返回新数组的特性
    }).reduce((ele,item)=>{
        return ele+item;
    },0)
}
