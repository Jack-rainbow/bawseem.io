4.转换规则

  只有当类型为Array,Object,Function才会只有ToPrimitive
  如何实现一个Toprimitive？
  如何给Toprimitive传入第二个转换的参数？
  当进行转换的时候是否是一直返回到原始值的时候才return？ 
    在转换的过程中首先进行ToPrimitive(val,PreferredType)转换,当PreferredType没有制定参数时，默认第一步会先进行number转换，然后在进行toString和valueof转换
  

  1.隐士类型转换里边有4种类型转换规则
  
    toNumber,toPrimitive,toString,toBoolean
    1.toSting ---只针对默认情况
      String([1,2,3,4]) ---> '1,2,3,4'
      String(null) ---> 'null'
      String(undefined) --->   'undefined'
      String(10) ---> '10'
      String({}) ---> '[object object]'
      String(NaN) ---> 'NaN'

    总结：通过string转换为除{},其他均为自身

    2.toNumber
      Number(null)  --->  0
      Number(undefined)  ---> NaN
      Number(NaN)  --->  NaN
      Number('10')  ---> 10
      Number('10a')  ---> NaN 
      Number(false)  ---> 0
      Number(true)  ---> 1
      Number([])  ---> 0
      Number({})  ---> NaN

    总结:
      1.通过Number转换为NaN的有undefined，'10a',{}.为什么是NaN？

    3.toBoolean

      Boolean(null)   --->  false
      Boolean(undefined) ---> false
      Boolean('1') ---> true
      Boolean(1) --->true
      Boolean([]) ---> true
      Boolean({}) ---> true
      Boolean(NaN) --->false
      Boolean('10a') --->true
      Boolean('') ---> false

    总结：
      1.当值为 false,undefined,null,NaN,'',+0,-0的时候为false，其他都为true
    
    4.toPrimitive-对象/数组转换为原始类型的操作
      
      当对象要转换为原始类型的时候，首先会去执行对象的valueOf方法，如果valueof返回原始类型的值(undefined，null，string，number，boolean)是原始类型的话，则toPrimitive的结果就是这个值
      如果valueOf不存在或者valueOf返回的不是原始类型的值，那么就会去调用Tostring值

    注意：不同的类型优先调用的方法是不同的，例如Date，优先调用toString方法

    总结：
      当值为对象类型的时候执行toPrimitive,如果是对象的话首先去调用valueOf，如果valueOf返回的不是原始类型则去调用toString，当toString返回的也不是原始类型的时候则抛错误。(如果没有返回返回原始类型是会一直查找的，直到找不到返回错误信息)


    对象类型在tonumber时会先toPrimitive？？？如何理解？

      当转换Number的时候会先执行toPrimitive,因为valueOf()返回自身，所以调用toString，返回""("" == false),然后去执行Number转换，得到0

      由此可以看出在类型进行的时候，会优先执行toPrimitive，然后在去执行相应的类型转换。是否正确？？？？？
    2.{}进行转换
      {}进行原始值转换的过程中是回返回toPrimitive,且{}原始值是[object object]

8.toPrimtive实现

var toPrimtive = function (obj,preferrdType = 'Number'){
  let utils = {
    typeOf:(obj)=>{
      return Object.prototype.toString.call(8,-1); //返回类型
    },
    isPrimtive:(obj)=>{
      let types = ['Null','undefined','String'.'Number','Boolean'];
      return types.indexOf(obj) !== -1;
    }
  };
  if(utils.isPrimitive(obj)){
    return obj;
  }
  preferredType = (preferredType == 'String' || utils.types(obj) == 'Date')?'String':'Number' ;
  if(utils.isPrimtive){
    if(utils.isPrimitive(obj).valueOf()){
      return obj.valueOf();
    }else{
      return obj.toString();
    }
  }else{
    if(utils.isPrimitive(obj).toString()){
      return obj.toString();
    }else{
      return obj.valueOf();
    }
  }
}
