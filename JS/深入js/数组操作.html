# 数组去重

方法一：遍历数组，建立新数组，利用indexOf判断是否存在于新数组中，不存在则push到新数组，最后返回新数组

    function delre(ar) {
    
        var ret = [];
    
        for (var i = 0, j = ar.length; i < j; i++) {
    
            if (ret.indexOf(ar[i]) === -1) {
    
                ret.push(ar[i]);
    
            }
    
        }
    
        return ret;
    
    }

# 获取数组中最大的值

Math.max 因为math.max参数只接受字符串，所以此处应使用apply

Math.max.apply(null,[]) //只适用于一位数组

因为在js中每一个Functin()函数都有call,apply方法 故Math.max()是函数，所以拥有此方法，所以利用此特性进行求解最大值

Math.max.apply(null,[[[]],[]].join(',').split(','))

# toString和toLocalString的区别

传入的值 toString toLocaleString
数字 字符串 字符串
字符串 字符串 字符串
数组 字符串 字符串
普通对象 “[object Object]”

正常用法上边都一样，但是当转换时间对象时，会出现问题 eg： (new Date).toString(); 

    (new Date).toLocaleString();
    <!--"2017/11/6 下午1:03:12"-->

同时千分位分割可以利用toLocalString()特性进行分割

(12312313123353).toLocalString('en-US')

	const obj = {};
	if (typeof keys === 'string') {
	keys = JSON.parse(keys.replace(/\w+/g, '"$&"'));
	}
	
	const iterate = (values, keys) =>

	keys.forEach((key, i) => {
		if(Array.isArray(key)) iterate(values[i], key)

	})

# 数组比差

const contrast = (ele,item) =>{
	const S_A = new Set(ele),S_B = new Set(item);
	return [...ele.filter(v=>!S_A.has(v)),item.filter(v=>!S_B.has(v))]
}
