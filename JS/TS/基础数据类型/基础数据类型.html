# 基础数据类型

数组类型

1.let arr:Array<number> = []
2.let arr:(string|number)= []
3.let arr:number[]

数据类型

1.let  str:string = '123'

元组类型(数组类型)

1.说明

必须严格按照格式书写

let arr: [string, number, boolean];
arr = ['1',2,true]

枚举类型

1.说明

类似对象，键值是默认是0-2

enum Roles{
    userAdmin,
    userImg,
    user
}

any类型

表示任意类型

let  value:any

void类型

表示什么都不是的类型

一般函数不返回值，要给函数指定类型的话就可以使用void类型

function abc():number --->此处的type表示函数return出来的值

object类型

对象类型

never 类型

永远不存在值的类型，例如报错信息，死循环的

类型断言

强行把一种类型转换为另外一个类型

1.写法1:

(<string>str)

2.写法2:
(str as string)

使用常量声明的时候必须赋值

使用函数没有return的时候可以使用void类型
