# 函数类型

1.说明

let  args:(a:number,b:number) => number //这表示一个完整的函数类型

2.可选参数

可选参数必须在必选参数后边写


x ?= number //表示可选参数

3.类型别名-type type使用别名

//?:表示可选参数,前边是必选参数
type add = (x: number, y: number, n?: number) => number;
let ddd: add;
ddd = (x:number,y:number):number => x+y

4.TS重载

i.说明 当传入不同类型及不同参数，返回是不同类型的时候可以用函数重载

ii.备注

函数重载只能用function定义

function abc(x: number): number[]//重载1
function abc(x: string): string[]//重载2
