# 泛型

1.说明

可以利用泛型创建多种数据类型的数据。

2.例如

//T可以传入任意类型
const abc = <T> (args:T):T=>{
    return args;
}
//传入string类型，返回string类型
console.log(abc<string>('1'));

3.泛型的类型推论

根据传入的参数自动帮助我们确定T类型
let output = identity("myString");//不推荐使用，不严谨

4.泛型变量 可以在参数里边定义具体泛型的类型

const abc = <T> (args:Array<T>):Array<T>=>{
    console.log(args.length);
    return args;
}
console.log(abc<string>([]));

let myIdentity: {<T>(arg: T): T} = abc; //使用赋值泛型变量
//另外一个写法
interface abcd {
    <T>(args:T):T;
}
let myIdentity: abcd = abc;
