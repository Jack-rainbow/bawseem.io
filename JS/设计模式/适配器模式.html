# 适配器模式

适配器则要把一个接口转换为另一个接口，她并不滤除某些能力，也不会简化接口。 作用

  • 1.适配2个插件库之间的区别
  • 2.避免修改客户现有代码

class A{
    d(){
        return '外国货';
    }
}
class B{
    constructor(){
        this.e = new A();
    }
    f(){
        let g = this.e.d();
        return `${g} +'转换器'+ '中国货'`;
    }
}
let q = new B();
console.log(q)
